#

### installation

```git
$ git clone https://github.com/fxhxyz4/comments-scraper.git
$ cd comments-scraper
```

#

### dependencies

```
$ npm install
```

#

### create API KEY

- [google console](https://console.cloud.google.com/apis/dashboard)

#

### webpack

```
$ npm run webpack
```
